## 一、产品概述

### 1.1 产品定位

**植物连连看** 是一款植物主题的堆叠式消除小游戏。

**核心玩法：** 点击可拾取的植物卡牌 → 放入下方卡槽 → 凑齐 3 张相同消除 → 消除所有卡牌通关。

**产品特色：**
- 植物主题卡牌（12 种植物）
- 极简玩法（3 秒上手）
- 断崖式难度（2 关设计）
- 广告变现（激励视频 + 插屏）

### 1.2 目标用户

| 用户群体 | 特征 | 占比 |
|----------|------|------|
| 休闲游戏玩家 | 碎片时间，喜欢简单游戏 | 60% |
| 植物爱好者 | 对植物感兴趣，喜欢收集 | 20% |
| 挑战型玩家 | 喜欢高难度，追求通关 | 15% |
| 其他 | 被话题吸引来的用户 | 5% |

### 1.3 核心指标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 次日留存 | ≥40% | 第一天玩，第二天还来 |
| 7 日留存 | ≥15% | 长期吸引力 |
| 人均游戏时长 | ≥5 分钟 | 单次游玩时长 |
| 人均广告观看 | ≥3 次 | 每人看广告次数 |
| 通关率（第二关） | ≤0.5% | 保持高难度 |
| **开发成本** | **0 元** | 零预算 |

## 二、玩法规则

### 2.1 基础规则

**三条核心规则：**

```
1. 点击卡牌
   - 只有最上层（未被覆盖）的卡牌可以点击
   - 被覆盖的卡牌点击后会抖动，无法拾取

2. 放入卡槽
   - 点击后卡牌移动到下方卡槽
   - 卡槽最多容纳 7 张卡牌

3. 消除规则
   - 卡槽内凑齐 3 张相同植物，自动消除
   - 消除后卡槽自动向左对齐
```

**胜利条件：** 消除所有卡牌

**失败条件：** 卡槽满 7 张且无法消除

### 2.2 卡牌堆叠规则

**层级关系：**

```
- 卡牌分多层堆叠（第一关 3 层，第二关 10 层）
- 上层卡牌覆盖下层卡牌
- 只有最上层的卡牌可点击

覆盖判断：
- 上层卡牌的 4 个角覆盖下层卡牌的中心区域
- 具体：行差∈[-1,0] 且 列差∈[-1,0]
```

**视觉表现：**

```
- 可点击卡牌：正常亮度
- 被覆盖卡牌：变暗（亮度 70%）
- 最上层卡牌：轻微高亮
```

### 2.3 卡槽规则

**卡槽设计：**

```
容量：7 格
尺寸：每格 40x50px
间距：5px
位置：屏幕底部中央（距底 80px）
```

**消除逻辑：**

```
1. 卡牌进入卡槽
2. 检查是否有 3 张相同植物
3. 如有，立即消除并腾出空间
4. 剩余卡牌自动向左对齐
```

**失败判定：**

```
- 第 7 张牌放入卡槽
- 无法与任何牌组成 3 张相同
- 游戏结束，显示失败界面
```

### 4.3 卡牌生成逻辑

```javascript
/**
 * 生成关卡卡牌
 */
function generateCards(level) {
  const config = LEVEL_CONFIG[level];
  const cards = [];
  
  // 1. 生成牌组（每组 3 张相同）
  const groups = config.totalCards / 3;
  const cardTypes = [];
  
  for (let i = 0; i < groups; i++) {
    const type = Math.floor(Math.random() * config.cardTypes) + 1;
    cardTypes.push(type, type, type); // 每组 3 张
  }
  
  // 2. 如果是死局模式，70% 概率移除一张牌
  if (config.deadEndChance > 0 && Math.random() < config.deadEndChance) {
    const removeIndex = Math.floor(Math.random() * cardTypes.length);
    cardTypes.splice(removeIndex, 1); // 移除一张，制造死局
  }
  
  // 3. 打乱顺序
  shuffleArray(cardTypes);
  
  // 4. 分配到层级位置
  const positions = generateLayerPositions(config);
  for (let i = 0; i < cardTypes.length; i++) {
    cards.push({
      id: `card_${i}`,
      type: cardTypes[i],
      layer: positions[i].layer,
      row: positions[i].row,
      col: positions[i].col,
      status: CardStatus.CLICKABLE,
      rotation: (Math.random() - 0.5) * 10, // 随机旋转 -5~5 度
      scale: 0.95 + Math.random() * 0.1     // 随机缩放 0.95~1.05
    });
  }
  
  // 5. 计算覆盖关系
  updateCardCoverage(cards);
  
  return cards;
}
```

## 五、道具系统

### 5.1 道具列表

**4 种基础道具：**

| 道具 | 图标 | 功能 | 使用场景 |
|------|------|------|----------|
| **移出** | 📤 | 移除卡槽前 3 张牌 | 卡槽快满时救急 |
| **洗牌** | 🔀 | 打乱所有未消除的牌 | 局面僵化时 |
| **撤回** | ↩️ | 撤销上一步操作 | 手滑时 |
| **复活** | ❤️ | 失败后清空卡槽继续 | 游戏结束时 |

### 5.2 道具获取

**2 种获取方式：**

```
1. 观看广告（30 秒激励视频）
   - 获得 1 个指定道具
   - 无次数限制

2. 分享（分享到微信群/朋友圈）
   - 获得 1 个随机道具
   - 每日限 3 次
```

**初始道具：**

```
新玩家首次进入第 2 关，赠送：
- 移出 x1
- 洗牌 x1
- 复活 x1
```

### 5.3 道具使用逻辑

```javascript
// 移出道具：移除卡槽前 3 张牌
function useRemove() {
  if (game.props.remove <= 0) return;
  if (game.slotCards.length === 0) return;
  
  const toRemove = game.slotCards.slice(0, 3);
  
  // 播放返回动画
  toRemove.forEach(card => {
    animateReturnToDeck(card);
    card.status = CardStatus.HIDDEN;
  });
  
  game.slotCards = game.slotCards.slice(3);
  game.props.remove--;
  
  // 检查消除（移除后可能触发）
  checkElimination();
}

// 洗牌道具：打乱所有未消除的牌
function useShuffle() {
  if (game.props.shuffle <= 0) return;
  
  // 收集所有未消除的牌
  const remaining = game.cards.filter(c => 
    c.status !== CardStatus.ELIMINATED && 
    c.status !== CardStatus.SELECTED
  );
  
  // 打乱类型
  const types = remaining.map(c => c.type);
  shuffleArray(types);
  
  // 重新分配
  remaining.forEach((card, i) => {
    card.type = types[i];
    updateCardVisual(card);
  });
  
  game.props.shuffle--;
}

// 撤回道具：撤销上一步
function useUndo() {
  if (game.props.undo <= 0) return;
  if (game.moveHistory.length === 0) return;
  
  const lastMove = game.moveHistory.pop();
  
  // 从卡槽移除
  const slotIndex = game.slotCards.findIndex(c => c.id === lastMove.cardId);
  game.slotCards.splice(slotIndex, 1);
  
  // 恢复卡牌状态
  const card = game.cards.find(c => c.id === lastMove.cardId);
  card.status = CardStatus.CLICKABLE;
  animateReturnToPosition(card, lastMove.x, lastMove.y);
  
  game.props.undo--;
}

// 复活道具：失败后清空卡槽
function useRevive() {
  if (game.props.revive <= 0) return;
  if (!game.isGameOver) return;
  
  // 清空卡槽
  game.slotCards.forEach(card => {
    card.status = CardStatus.HIDDEN;
  });
  game.slotCards = [];
  
  // 恢复游戏状态
  game.isGameOver = false;
  hideGameOverPanel();
  
  game.props.revive--;
}
```

## 六、广告系统

### 6.1 广告类型

**2 种广告：**

| 类型 | 触发场景 | 广告时长 | eCPM 估算 |
|------|----------|----------|----------|
| **激励视频** | 获取道具、复活 | 30 秒 | 80-150 元 |
| **插屏广告** | 关卡结束（可选） | 5-10 秒 | 30-60 元 |

### 6.2 广告触发点

**激励视频广告：**

```
触发点 1：获取道具
- 点击道具按钮（道具数量为 0 时）
- 弹出广告提示："观看视频获得 1 个道具"
- 用户确认后播放广告
- 广告结束后发放道具

触发点 2：失败复活
- 游戏失败时，弹出复活提示
- "观看视频复活，继续挑战"
- 用户确认后播放广告
- 广告结束后清空卡槽，继续游戏
```

**插屏广告：**

```
触发点：关卡结束
- 通关后 2 秒，自动弹出插屏广告
- 可关闭（X 按钮）
- 不强制观看
```

### 6.3 广告代码接入

```javascript
// 广告管理器
class AdManager {
  constructor() {
    this.adUnits = {
      rewardVideo: 'adunit-xxxxxxxxxxxxx',  // 激励视频
      interstitial: 'adunit-yyyyyyyyyyyyy'  // 插屏广告
    };
    this.rewardedVideoAd = null;
    this.pendingReward = null;
    
    this.init();
  }
  
  init() {
    // 初始化激励视频广告
    if (wx.createRewardedVideoAd) {
      this.rewardedVideoAd = wx.createRewardedVideoAd({
        adUnitId: this.adUnits.rewardVideo
      });
      
      this.rewardedVideoAd.onLoad(() => {
        console.log('激励视频加载成功');
      });
      
      this.rewardedVideoAd.onError((err) => {
        console.error('激励视频加载失败', err);
      });
      
      this.rewardedVideoAd.onClose((res) => {
        if (res && res.isEnded) {
          // 完整观看，发放奖励
          this.grantReward();
        } else {
          // 中途退出
          showTip('未完整观看，无法获得奖励');
        }
      });
    }
  }
  
  // 观看广告获得道具
  watchAdForProp(propType) {
    this.pendingReward = { type: 'prop', propType: propType, count: 1 };
    
    if (this.rewardedVideoAd) {
      this.rewardedVideoAd.show().catch(() => {
        this.rewardedVideoAd.load().then(() => {
          this.rewardedVideoAd.show();
        });
      });
    }
  }
  
  // 发放奖励
  grantReward() {
    const reward = this.pendingReward;
    
    if (reward.type === 'prop') {
      game.addProp(reward.propType, reward.count);
      showTip(`获得 ${reward.count} 个${getPropName(reward.propType)}！`);
    }
    
    this.pendingReward = null;
  }
}
```

// Card.js - 卡牌类
class Card {
  constructor(config) {
    this.id = config.id;
    this.type = config.type;
    this.layer = config.layer;
    this.row = config.row;
    this.col = config.col;
    this.x = config.x;
    this.y = config.y;
    this.zIndex = config.zIndex;
    this.status = CardStatus.COVERED;
  }
  
  // 检查是否可点击
  isClickable() {
    return this.status === CardStatus.CLICKABLE;
  }
  
  // 点击处理
  onClick() {
    if (this.isClickable()) {
      game.onCardClick(this);
    } else {
      this.shake();
    }
  }
  
  // 抖动动画
  shake() {
    wx.createAnimation({
      duration: 300,
      timingFunction: 'ease-in-out'
    }).step()
      .translateX(-5).step()
      .translateX(5).step()
      .translateX(0).step()
      .export();
  }
}

// GameState.js - 游戏状态管理
class GameState {
  constructor() {
    this.currentLevel = 1;
    this.cards = [];
    this.slotCards = [];
    this.isAnimating = false;
    this.isGameOver = false;
    this.props = {
      remove: 1,
      shuffle: 1,
      undo: 1,
      revive: 1
    };
  }
  
  // 添加卡牌到卡槽
  addToSlot(card) {
    if (this.slotCards.length >= 7) return false;
    
    this.slotCards.push(card);
    card.status = CardStatus.SELECTED;
    
    // 检查消除
    this.checkElimination();
    
    // 检查失败
    if (this.slotCards.length === 7) {
      this.checkFailure();
    }
    
    return true;
  }
  
  // 检查消除
  checkElimination() {
    const typeCount = {};
    this.slotCards.forEach(card => {
      typeCount[card.type] = (typeCount[card.type] || 0) + 1;
    });
    
    for (const [type, count] of Object.entries(typeCount)) {
      if (count === 3) {
        this.eliminateCards(parseInt(type));
        return true;
      }
    }
    return false;
  }
  
  // 消除卡牌
  eliminateCards(type) {
    const toEliminate = this.slotCards.filter(c => c.type === type);
    
    // 播放消除动画
    toEliminate.forEach(card => {
      card.status = CardStatus.ELIMINATED;
    });
    
    // 从卡槽移除
    this.slotCards = this.slotCards.filter(c => c.type !== type);
    
    // 检查胜利
    this.checkVictory();
  }
  
  // 检查胜利
  checkVictory() {
    const remaining = this.cards.filter(
      c => c.status !== CardStatus.ELIMINATED
    );
    
    if (remaining.length === 0) {
      this.onVictory();
    }
  }
}
