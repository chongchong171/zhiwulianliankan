## 二、广告接入详细方案

### 📋 流量主开通条件

**官方要求（2025 年最新）：**

```
条件一（满足任一即可）：
✅ 小游戏累计独立访客（UV）> 500
✅ 无刷粉行为，且未曾有严重违规记录

条件二（同公司主体）：
- 名下已有小游戏
- 该小游戏满足条件一
- 持续 1 个季度有广告变现收入
- 历史不存在流量主违规记录
```

**开通流程：**

```
第 1 步：注册小游戏账号
  └─ 微信公众平台 (mp.weixin.qq.com)
  └─ 注册类型：小游戏
  └─ 主体类型：个人/企业

第 2 步：开发并上架小游戏
  └─ 完成开发
  └─ 提交审核
  └─ 审核通过后发布

第 3 步：积累用户（>500 UV）
  └─ 自然增长 或
  └─ 投放广告（100 元约可获得 600-1000 用户）

第 4 步：申请流量主
  └─ 登录微信公众平台
  └─ 左侧菜单：流量主
  └─ 同意协议，快速开通

第 5 步：创建广告位
  └─ 选择广告类型（激励视频、插屏等）
  └─ 输入广告位名称
  └─ 获取广告位代码（adUnitId）

第 6 步：代码接入
  └─ 将 adUnitId 填入游戏代码
  └─ 测试广告展示
  └─ 提交广告位审核

第 7 步：获得收入
  └─ 广告审核通过
  └─ 用户观看广告
  └─ T+1 日查看收入数据
```

### 💰 广告类型与收益

**微信小游戏支持的广告类型：**

| 类型 | 展示形式 | eCPM（元/千次） | 适用场景 |
|------|----------|----------------|----------|
| **激励视频** | 用户主动观看 30s 视频 | 80-150 | 获取道具、复活 |
| **插屏广告** | 关卡切换时弹出 | 30-60 | 过关/失败界面 |
| **Banner 广告** | 底部条幅 | 5-15 | 不推荐（影响体验） |
| **格子广告** | 原生模板 | 20-40 | 道具商店 |

**收益估算：**

```
假设日活 1000 人：
- 人均观看激励视频：3 次
- 激励视频 eCPM：100 元
- 日广告展示：1000 × 3 = 3000 次
- 日收入：3000 / 1000 × 100 = 300 元
- 月收入：300 × 30 = 9000 元

分成比例：
- 普通小游戏：50%
- 创意小游戏：70%（200 万以内部分）

实际月收入（按 50% 分成）：4500 元/月
```

### 🔧 代码接入实现

**1. 初始化广告 SDK：**

```javascript
// game.js - 广告初始化
class AdManager {
  constructor() {
    this.adUnits = {
      rewardVideo: 'adunit-xxxxxxxxxxxxx',  // 激励视频
      interstitial: 'adunit-yyyyyyyyyyyyy'  // 插屏广告
    };
    this.rewardedVideoAd = null;
    this.interstitialAd = null;
    
    this.initAds();
  }
  
  initAds() {
    // 激励视频广告
    if (wx.createRewardedVideoAd) {
      this.rewardedVideoAd = wx.createRewardedVideoAd({
        adUnitId: this.adUnits.rewardVideo
      });
      
      this.rewardedVideoAd.onLoad(() => {
        console.log('激励视频广告加载成功');
      });
      
      this.rewardedVideoAd.onError((err) => {
        console.error('激励视频广告加载失败', err);
      });
      
      this.rewardedVideoAd.onClose((res) => {
        if (res && res.isEnded) {
          // 用户完整观看，发放奖励
          this.grantReward();
        } else {
          // 用户中途退出，不发放奖励
          this.showTip('未完整观看，无法获得奖励');
        }
      });
    }
    
    // 插屏广告
    if (wx.createInterstitialAd) {
      this.interstitialAd = wx.createInterstitialAd({
        adUnitId: this.adUnits.interstitial
      });
    }
  }
}
```

**2. 激励视频广告（获取道具）：**

```javascript
// 观看广告获得道具
function watchAdForProp(propType) {
  const adManager = game.adManager;
  
  if (adManager.rewardedVideoAd) {
    adManager.rewardedVideoAd.show()
      .catch(() => {
        // 失败重试
        adManager.rewardedVideoAd.load()
          .then(() => adManager.rewardedVideoAd.show());
      });
    
    // 设置期望获得的道具类型
    adManager.pendingReward = {
      type: 'prop',
      propType: propType,
      count: 1
    };
  } else {
    showTip('广告功能暂未开通');
  }
}

// 发放奖励
function grantReward() {
  const reward = game.adManager.pendingReward;
  
  if (reward.type === 'prop') {
    game.addProp(reward.propType, reward.count);
    showTip(`获得 ${reward.count} 个${getPropName(reward.propType)}！`);
  }
  
  // 上报数据
  reportAdReward(reward);
}
```

```javascript
class AdCooldown {
  constructor() {
    this.lastShowTime = 0;
    this.cooldownMs = 30000; // 30 秒冷却
  }
  
  canShow() {
    const now = Date.now();
    if (now - this.lastShowTime >= this.cooldownMs) {
      this.lastShowTime = now;
      return true;
    }
    return false;
  }
}
```

## 三、差异化优化方案

### 🚀 优化方向（不是降级）

**羊了个羊的问题：**

| 问题 | 用户反馈 | 我们的优化 |
|------|----------|------------|
| ❌ 第二关太难 | "根本过不去，纯看运气" | 渐进难度 + 保底机制 |
| ❌ 广告太多 | "不看广告完全玩不了" | 减少强制广告、增加获取途径 |
| ❌ 没有成长 | "玩了很久还是什么都没有" | 收集系统 + 成就系统 |
| ❌ 内容单一 | "只有两关，很快就腻了" | 多关卡 + 每日挑战 |
| ❌ 没有知识含量 | "纯浪费时间" | 植物知识植入 |

### 📈 具体优化方案

#### 1. 难度曲线优化

**羊了个羊的模式：**
```
关卡 1：难度 1/10（不可能输）
关卡 2：难度 10/10（不可能赢）← 断崖式难度
```

**我们的优化：**
```
关卡 1：难度 1/10（教学关）
关卡 2：难度 3/10（简单）
关卡 3：难度 5/10（中等）
关卡 4：难度 7/10（困难）
关卡 5：难度 9/10（挑战）
每日挑战：难度 10/10（地狱）
```

#### 2. 道具获取优化

**羊了个羊的模式：**
```
道具获取：
- 看广告（30 秒）
- 分享（拉人头）

```

**我们的优化：**
```

✅ 成就奖励 - 达成目标获得
✅ 植物识别 - AI 管家联动
✅ 分享（可选）- 不强制
✅ 游戏内货币 - 通关获得金币购买
→ 6 种获取方式，不强制看广告
```

**实现代码：**

```javascript
// 多种道具获取方式
class PropSystem {
  // 1. 每日签到
  dailyCheckIn() {
    if (!this.hasCheckedInToday()) {
      this.addProp('shuffle', 2);
      this.addProp('remove', 2);
      this.markCheckedIn();
      showTip('签到成功！获得 2 个洗牌 +2 个移出');
    }
  }
  
  // 2. 成就奖励
  checkAchievements() {
    const achievements = [
      { id: 'pass_level_3', condition: () => game.maxLevel >= 3, reward: { prop: 'revive', count: 3 } },
      { id: 'collect_12_types', condition: () => game.collectedTypes >= 12, reward: { prop: 'undo', count: 5 } }
    ];
    
    for (const ach of achievements) {
      if (!this.isUnlocked(ach.id) && ach.condition()) {
        this.unlock(ach.id);
        this.addProp(ach.reward.prop, ach.reward.count);
      }
    }
  }
  
  // 3. 游戏货币购买
  buyProp(propType, cost) {
    if (game.coins >= cost) {
      game.coins -= cost;
      this.addProp(propType, 1);
      reportPurchase('prop', propType, cost);
    }
  }
}
```

```javascript
    receiveHelp: function() {
      // 接收好友赠送的道具
      const gifts = getPendingGifts();
      for (const gift of gifts) {
        addProp(gift.type, gift.count);
      }
    }
  },
  
  // 3. 植物交换（特色社交）
  plantExchange: {
    request: function(plantId) {
      // 向好友请求交换植物卡牌
      sendExchangeRequest(friendId, plantId);
    },
    fulfill: function(requestId, plantId) {
      // 满足好友的请求
      fulfillExchange(requestId, plantId);
    }
  }
};
```
